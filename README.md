# 🚗 Bengaluru Traffic Simulator & Router

An interactive web application that predicts traffic volume, average speed, and travel times in Bengaluru using Machine Learning. Built with Python, `scikit-learn` (Decision Trees), `Folium` for dynamic mapping, and an interactive front-end using `Streamlit`. 

## 🌟 Features

* **🗺️ Live Route Map:** Visualizes source areas and destination roads/intersections with interactive map markers and accurate red polylines fetched via real routing APIs (OSRM).
* **🕰️ Simulation Controls:** Simulate traffic by selecting Start Area, Destination, Weather Conditions, and Roadwork activity. Fine-tune parameters like Congestion Level, Road Capacity Utilization, Travel Time Index, Incident Reports, and more.
* **🚦 Real-Time Predictions:** Instantly outputs Predicted Traffic Volume, Average Speed, Mapped Route Distance, and Estimated Time Taken. Includes color-coded volume indicators (Low, Medium, High).
* **📥 Dataset Integration:** Directly download the curated raw dataset used to train the machine learning models.

## 💻 Tech Stack & Libraries

- **Languages:** Python
- **Machine Learning & Data Processing:** 
  - `scikit-learn` (DecisionTreeRegressor for modeling, OneHotEncoder for categorical data)
  - `pandas` (Data manipulation)
- **Frontend / UI:** 
  - `streamlit` (Web app framework)
  - `streamlit.components.v1` (Rendering HTML maps inside the app)
- **Maps & Geocoding:**
  - `folium` (Interactive map generation)
  - `geopy` (Nominatim for location geocoding and geodesic distance calculation)
  - `requests` (Fetching driving routes from the OSRM API)
- **Serialization:** `pickle` (Saving/loading trained models and distance mappings)

## 📊 About the Dataset

The project utilizes the `Banglore_traffic_Dataset.csv` dataset. 
- **Categorical Features:** Area Name, Road/Intersection Name, Weather Conditions, Roadwork and Construction Activity.
- **Numerical Features:** Congestion Level (%), Road Capacity Utilization (%), Travel Time Index, Incident Reports, Public Transport Usage, Parking Usage, Pedestrian and Cyclist Count.
- **Target Variables:** Traffic Volume and Average Speed. 
The dataset captures the complex dynamics of Bengaluru's traffic, allowing the model to understand the relationship between environmental factors, road conditions, and actual traffic flow.

## 🧠 Model and Mapping Insights

- **The Model:** Dual `DecisionTreeRegressor` models are trained simultaneously—one to predict the total Traffic Volume and another to predict the Average Speed. The model handles unseen categories gracefully and caps depths to prevent overfitting.
- **The Maps:** The map generation script uses `geopy` to find exact coordinates of Bengaluru locations (with hardcoded fallbacks for accuracy). It then queries the public `OSRM` routing API to draw the actual road paths (polylines) between areas and destinations, rather than just straight lines, providing highly realistic map visualizations.

## 🚀 Setup & Installation

Follow these instructions to get a copy of the project up and running on your local machine.

### 1. Prerequisites & Cloning
Make sure you have Python 3.8+ installed on your system. First, clone the repository or download the project files to your local machine.

Navigate to the project folder and install the required Python modules:
```bash
pip install pandas scikit-learn streamlit folium geopy requests
```

### 2. Train the Machine Learning Model
The application relies on pre-trained models to deliver fast predictions. Run the training script to parse the dataset and generate `model.pkl`:
```bash
python train_model.py
```

### 3. Generate the Map and Distances
Next, generate the geographic data, routing paths, and the interactive HTML map. This will create `distances.pkl` and `bangalore_traffic_map.html`:
```bash
python generate_map.py
```
*Note: This script might take a minute or two as it respects rate limits for the geocoding and routing APIs.*

### 4. Run the Web Application
Once both `.pkl` files and the HTML map are successfully generated, launch the Streamlit server:
```bash
python -m streamlit run app.py
```

Streamlit will automatically open your default browser pointing to `http://localhost:8501`.
